# Orbit Luxuries - Premium E-Commerce Platform

A high-performance, full-stack luxury e-commerce application built with Next.js, featuring a custom storefront and a comprehensive Admin Dashboard.

## 🚀 Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Database & Auth:** Firebase (Firestore & Authentication)
* **Icons:** Lucide React
* **State Management:** React Context API

## ✨ Key Features
* **Dynamic Catalog:** Live-synced product listings with advanced filtering by category, brand, and price.
* **Premium UI/UX:** Tata CLiQ-inspired mega menus, A-Z brand sorting, and high-end responsive design.
* **Secure Admin Panel:** Protected route accessible only by authorized administrators.
* **Command Dashboard:** Real-time metrics, order management, and live inventory control.
* **Firebase Integration:** Seamless data synchronization for Products, Orders, Banners, and User Profiles.
